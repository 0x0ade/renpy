[edit] 操作方法

ゲームを読み進めるには、左クリックか、スペースキー、エンターキーを押してください。メニューが表示された場合、左クリックで選択するか、方向キーで選択してエンターキーで決定してください。

[edit] ゲームメニュー

ゲーム中に右クリックか Esc を押すと、ゲームメニューを開きます。ゲームメニューでは以下の操作をすることができます。

戻る
ゲームに戻ります。
セーブ
セーブスロットをクリックすることで、現在の状態を保存できます。
ロード
セーブスロットをクリックすることで、データをロードできます。 ”オート”をクリックするとオートセーブされたデータをロードすることができます。
環境設定
ゲームの環境設定を変更することができます。
画面表示
フルスクリーンとウインドウモードを切り替えます。
画面効果
画面切替時の効果の有無を選択できます。
文字表示速度
文字表示速度を変更します。スライダーを右に動かすほど早く、左に動かすほど遅く表示されます。右端に動かすと、瞬間的に表示されます。
ジョイスティック
ジョイスティックのボタンを割り当てます。
スキップ
既読メッセージのみをスキップするか、未読もスキップするか選択します。
スキップ開始
ゲームに戻り、スキップを開始します。
選択肢後
選択肢後にスキップを続けるかどうか選択します。
オート待ち時間
オートで読み進める待ち時間を設定します。スライダーを左に動かすほど、読み進める待ち時間が短くなります。右端に動かすと、オートで読み進めるのを止めます。
音楽、効果音、ボイス音量
音楽、効果音、ボイスの各音量を設定出来ます。スライダーを右へ動かすほど、音が大きくなります。
メインメニュー
現在のゲームを終了して、メインメニューに戻ります。
ヘルプ
ヘルプ画面を表示します。
終了
ゲームを全て終了して、ウインドウを閉じます。

[edit] キーボードとマウスの割り当て

左クリック、エンターキー
ゲームを進めたり、メニューやボタン、スライダーを決定します。
スペースキー
ゲームを進めますが、メニューやボタン、スライダーを決定できません。

方向キー
メニューやボタン、スライダーを選択します。
Ctrl
キーを押している間、ゲームをスキップします。.
Tab
一度押すとスキップを開始します。もう一度押すとスキップを停止します。
マウスホイール上、PageUp
ロールバックを行い、ゲームを巻き戻します。テキストを再び表示したり、別の選択肢を選んだりできるようになります。
マウスホイール下、PageDown
ロールフォワードを行い、ロールバック前の状態に戻ります。
右クリック、Esc
ゲームメニューを表示します。ゲームメニュー表示中は、ゲームに戻ります。
中クリック、H
テキストウィンドウやその他の表示を隠します。
F
フルスクリーンモードを切り替えます。
S
スクリーンショットを撮ります。スクリーンショットは screenshotxxxx.png  というファイル名で保存されます。
Alt-M, Command-M
ウインドウを最小化（アイコン化）します。
Alt-F4, Command-Q
ゲームを終了します。
Delete
選択しているセーブスロットを消去します。
